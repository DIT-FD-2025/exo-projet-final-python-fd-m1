# Projet Final : Analyse et Visualisation de Donnees

## Presentation du projet

L'objectif de ce projet est de realiser une etude complete, de l'exploration des donnees brutes jusqu'a la mise en ligne d'un outil de visualisation interactif.

Ce travail permet de demontrer la maitrise de la chaine de traitement de donnees vue en cours.

## Missions a realiser

### 1. Exploration et Preparation (Pandas)

Objectif : Nettoyer et structurer le jeu de donnees pour l'analyse.

Points traites dans le notebook :
- Chargement et inspection des donnees
- Nettoyage
- Manipulation et preparation des variables

### 2. Analyse Graphique

Objectif : Identifier des tendances et des correlations visuelles.

Points traites dans le notebook :
- Statique : utilisation de Matplotlib et Seaborn pour les analyses de fond
- Interactif : utilisation de Plotly Express pour une exploration dynamique des donnees

### 3. Dashboard Interactif (Streamlit)

Objectif : Creer une interface utilisateur conviviale.

Points traites dans l'application :
- Conception d'un tableau de bord avec Streamlit
- Integration de filtres et de graphiques interactifs

### 4. Deploiement

Objectif : Rendre le travail accessible a tous.

Preparation incluse dans le projet :
- Configuration pour un usage en local
- Preparation du deploiement sur Streamlit Cloud ou autre

## Fichiers du projet

- `Projet_Final_Analyse_Donnees.ipynb` : fichier `.ipynb` complet
- `app.py` : application Streamlit
- `Dataset.csv` : jeu de donnees
- `requirements.txt` : dependances necessaires

## Utilisation en local

Se placer dans le dossier du projet :

```bash
cd "Projet final"
```

Installer les dependances :

```bash
pip install -r requirements.txt
```

Lancer le Dashboard Interactif :

```bash
streamlit run app.py
```
